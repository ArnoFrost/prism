# skills/ — 技能模板层（系统层）

此目录保存 Prism 的技能模板和 schema 定义。

## 系统层 vs 实例层

- **系统层**（本目录）：技能的模板、schema、约束定义
- **实例层**（外部挂载）：实际的技能文件，存放在用户的 Skills 仓库中，通过挂载方式接入产物路径

## 挂载模式

技能实例的挂载与路书挂载模式一致：

```
产物路径/
├── skills.local  -> 用户的 Skills 仓库（软链接）
└── ai-task.local -> 用户的 AI-TASK vault（软链接）
```

Prism 仓库内不存放实际技能文件，只定义技能应遵循的结构规范。
