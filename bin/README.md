# bin/ — 工具入口

Prism 的可执行工具入口。每个脚本可配合同名 Skill 使用，形成"脚本 + 自然语言"的双通道能力。

## 规划工具

| 命令 | 职责 | 配对 Skill |
|------|------|-----------|
| `setenv` | 注入 `PRISM_DIR` 等环境变量到 shell profile | 待定 |
| `sync-skill` | 将技能模板同步到产物路径（类似路书挂载） | 待定 |
| `init-project` | 初始化一个项目接入 Prism（创建软链接、AGENT 文件） | 待定 |

## 设计约束

- 脚本应保持幂等（多次执行结果一致）
- 失败时应给出明确的错误信息而非静默跳过
- 路径参数优先从环境变量读取，fallback 到合理默认值
